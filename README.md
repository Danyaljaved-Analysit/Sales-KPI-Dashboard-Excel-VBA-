# Sales KPI Dashboard-Excel-VBA-
This project is a sales dashboard built in Excel to track and analyze performance across different regions .  I worked on organizing raw data, creating pivot tables, and designing a clean dashboard layout to make insights easy to understand. The dashboard allows filtering by region and other fields using slicers, which helps in quick analysis.  

**Key Features**
Interactive dashboard with slicers
KPI cards for Sales, Profit, and Quantity
Region/state-wise performance analysis
Clean and structured data handling
Basic automation using VBA macros

** Tools Used**
Microsoft Excel
Pivot Tables
Slicers
VBA (Macros)

 Notes:

This project was created as part of my learning journey in data analysis. I focused on making the dashboard simple, readable, and useful for basic business insights.
